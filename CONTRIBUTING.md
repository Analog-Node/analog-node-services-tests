These repositories document real-world experiments focused on bridging legacy
and analog systems to modern networks. Contributions from sysops, hobbyists, and
experimenters are welcome.

## What You Can Contribute

You may contribute:

- Test results for hardware or services
- Configuration notes or workarounds
- Corrections or clarifications
- Additional test cases
- Documentation improvements

All contributions should be factual, reproducible where possible, and clearly
documented.

## How to Contribute

1. Fork the repository
2. Create a new branch for your contribution
3. Add your content using the provided templates
4. Submit a pull request with a clear description

For test results, please use the standard **Test Report Template**.

## Style Guidelines

- Use Markdown
- Be concise and technical
- Avoid promotional language
- Clearly state hardware, software versions, and dates
- Note any assumptions or limitations

## Attribution

By contributing, you agree that your contribution will be licensed under the
MIT License, consistent with the rest of the repository.

## Questions

If you are unsure where something belongs or how to structure it, open an issue
and start a discussion before submitting a pull request.
