# Analog Node — Services Test Lab

This repository contains test protocols, documentation, and results related to
**network and service-based platforms** used with legacy and analog systems.

The focus is on evaluating VoIP, SIP, and fax services for compatibility with
analog modems, fax machines, and BBS-related workloads.

Bridging Analog Systems to Modern Networks.

## Scope

This repository covers testing and experimentation involving:

- SIP and VoIP service providers
- Fax-over-IP (FoIP) platforms
- T.38 and G.711 pass-through behavior
- Analog modem connectivity over IP
- Long-duration and stability testing

The goal is to determine practical suitability for sysop and hobbyist use.

## What You’ll Find Here

- Service test protocols and procedures
- Provider-specific test results
- Compatibility summaries
- Configuration notes and workarounds
- Templates for documenting new services

## Intended Audience

- BBS sysops
- VoIP and telephony experimenters
- Fax-over-IP testers
- Anyone bridging analog systems to IP networks
